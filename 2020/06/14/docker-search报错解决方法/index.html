docker search报错解决方法

发表于 2020-06-14 | 分类于 docker |
字数统计: 429 | 阅读时长 ≈ 2

解决方法:

通过dig @114.114.114.114 registry-1.docker.io找到可用ip

尝试修改/etc/hosts强制docker.io相关的域名解析到其它可用IP

保存后重试，成功！！
